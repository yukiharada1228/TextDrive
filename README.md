# TextDrive - 文字だけドライブゲーム

pygameを使用したシンプルなドライブゲームです。文字だけで構成されたコースを車で走り抜けます。

## ゲームの特徴

- 文字（■）で構成されたコース
- 左右の矢印キーで車を操作
- 壁にぶつかるとゲームオーバー
- 距離を競うシンプルなゲーム

## 必要な環境

- Python 3.12以上
- pygame 2.6.1以上

## インストール方法

1. リポジトリをクローン
```bash
git clone https://github.com/your-username/TextDrive.git
cd TextDrive
```

2. 依存関係をインストール
```bash
uv sync
```

## 実行方法

```bash
python main.py
```

## 操作方法

- 左矢印キー: 左に移動
- 右矢印キー: 右に移動
- Rキー: ゲームオーバー時にリスタート

## ゲームのルール

- 黒い四角（■）は壁で、ぶつかるとゲームオーバー
- 空白部分を通って進みます
- できるだけ長い距離を走り抜けましょう
